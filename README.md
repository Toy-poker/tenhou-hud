# 天鳳HUD

天鳳で対戦相手の統計データをリアルタイム表示するChrome拡張機能です。

## 表示項目

- 和了率 / 放銃率
- リーチ率 / 副露率
- 鳴き防御 / 染め手率
- ダマ率 / 安定段位

鳳凰卓全体の分布と比較して色で視覚的にフィードバックされます。

## 動作環境

- Google Chrome / Microsoft Edge
- Desktop4K版、Web版どちらでも動きます

## インストール方法

1. 右上の緑色の「Code」ボタン → 「Download ZIP」でダウンロード
2. ZIPを解凍する
3. Chromeの場合は `chrome://extensions/` 、Edgeの場合は `edge://extensions/` を開く
4. 右上の「デベロッパーモード」をオンにする
5. 「パッケージ化されていない拡張機能を読み込む」をクリック
6. 解凍したフォルダを選択

## 注意事項

- 鳳凰卓未経験のプレイヤーはデータが表示されません
- データの提供元は [nodocchi.moe](https://nodocchi.moe) です
